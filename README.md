# [Josh Ward's Dev Site](https://joshward.dev)

## Tech Stack

- [Next.js](https://nextjs.org/)
- [Typescript](https://www.typescriptlang.org/)
- [Tailwindcss](https://tailwindcss.com/)
- Hosted on [Vercel](https://vercel.com/)

## Commands

### Run

```bash
npm run dev
```

### Build

```bash
npm run build
```
